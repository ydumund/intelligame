# IntelliGame

类飞机大战小游戏

## 一句话概括

雷神扔锤子打怪物防止房子被毁

* 玩法:  
  * WSAD:上下左右  
  * 鼠标左键:扔锤子

## 游戏机制

玩家掉血后会有1秒的无敌时间

## 开发空间

* 杀怪充能然后释放大招
* 添加BOSS怪物种类
* 杀死BOSS后掉落回血物品
* 到达一定分数获得奖励
* ......
